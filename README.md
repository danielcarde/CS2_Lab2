Lab 2: Two-Dimensional Arrays

A Java data structures lab focused on reading and processing normal and jagged two-dimensional array data.

Highlights
Works with normal 2D arrays.
Works with jagged 2D array input.
Uses multiple local input files.
Practices nested-loop processing.

Overview
Lab 2 explores two-dimensional arrays using structured input files. The lab includes both normal rectangular arrays and jagged arrays, giving practice with different data layouts.
The program demonstrates how array shape affects parsing and iteration.

Why This Project
Two-dimensional arrays are common in matrix, grid, image, and table problems. This lab builds comfort with nested indexing before later data structure assignments.

Usage
Compile: Compile with javac *.java.
Run: Execute the program with java Lab2_Cardenas.
Keep input.txt, jagged_2d_array, and normal_2d_array.txt in the same folder.

Installation
Ensure you have the Java Development Kit (JDK) installed on your system.
Open a terminal in the lab folder before compiling so the program can find any required local text files.

Architecture
File Structure: The project contains Lab2_Cardenas.java and supporting text files for array input.
Components: The Java file handles parsing and processing. The text files provide array data.

Future Improvements
